1.  **Исправить ошибку `TypeError: unsupported operand type(s) for +: 'NoneType' and 'int'` при покупке предметов за Biocoin.**
    *   В файле `modules/handlers/items.py` в функциях обработки покупок (строки 59 и 191) происходит обновление поля `total_spent`. В базе данных `total_spent` может быть `None` (NULL), поэтому `u['total_spent'] + cost` вызывает ошибку.
    *   Нужно заменить `u['total_spent']` на `int(u.get('total_spent', 0) or 0)`.
    *   Найти все места в `modules/handlers/items.py` (или других файлах), где происходит подобное обновление `total_spent` и применить безопасное получение значения.

2.  **Исправить ошибку сортировки в рейтинге по Синдикату (spent).**
    *   В `database.py` функция `get_user_rank` для `sort_by='spent'` использует поле `total_spent`, которое может быть NULL.
    *   В запросе сравнения `WHERE (total_spent, xp) > ...` если `total_spent` NULL, то поведение может быть некорректным. Нужно обернуть в `COALESCE(total_spent, 0)`.
    *   Запрос должен выглядеть так: `WHERE (COALESCE(total_spent, 0), xp) > (SELECT COALESCE(total_spent, 0), xp FROM players WHERE uid = %s)`.
