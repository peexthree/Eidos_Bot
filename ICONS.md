# Документация иконок Эйдоса (Eidos SVG Icons)

Данный документ содержит реестр всех SVG иконок, используемых в проекте, и их предназначение.
Иконки расположены в `static/IMG/`.

## СЕКТОР 1: ПРОФИЛЬ И БАЗА (menu.py, user.py)
Эти глифы отвечают за идентичность пользователя и его прогресс.

- `eidos_demiurge-user.svg` (Демиург) -> Главный экран профиля, кнопка "Мой статус".
- `eidos_xp-rank.svg` (Ранг / XP) -> Индикатор текущего уровня и полоса прогресса.
- `eidos_streak-power.svg` (Стрик) -> Счетчик дней подряд (дисциплина).
- `eidos_skill-tree.svg` (Древо Навыков) -> Меню прокачки статов.
- `eidos_biometric-id.svg` (Биометрия) -> Команда `/dossier` или карточка игрока в базе.
- `eidos_neuro-brain.svg` (База Знаний) -> Доступ к лору, гайдам и справочнику Эйдоса.

## СЕКТОР 2: ИНВЕНТАРЬ И ЭКОНОМИКА (inventory.html, items.py, shop.py, crafting.py)
Интерфейс WebApp строится на этих узлах.

- `eidos_inventory-cache.svg` (Инвентарь) -> Главная иконка WebApp-инвентаря, кнопка "Рюкзак".
- `eidos_weapon-attack.svg` (Атака) -> Слот экипировки оружия, стат Силы/Атаки.
- `eidos_shield-armor.svg` (Броня) -> Слот экипировки брони, стат Защиты.
- `eidos_medkit-repair.svg` (Ремкомплект) -> Расходники для восстановления HP.
- `eidos_forge-craft.svg` (Крафт Кузня) -> Раздел `crafting.py` (создание предметов из лута).
- `eidos_credits-crypto.svg` (Кредиты) -> Базовая валюта (отображение баланса).
- `eidos_shard-premium.svg` (Кристалл) -> Премиум-валюта или редкие ресурсы с рейдов.
- `eidos_black-market.svg` (Черный Рынок) -> Вызов `shop.py` (торговля, покупка бустов).

## СЕКТОР 3: БОЕВАЯ СИСТЕМА И ИВЕНТЫ (pvp.py, raid.py, glitch_system.py)
Агрессивная кинетика для стрессовых модулей.

- `eidos_arena-pvp.svg` (PvP Арена) -> Вход в режим дуэлей между игроками.
- `eidos_sniper-aim.svg` (Снайпер) -> Выбор цели в PvP или использование спецприема.
- `eidos_target-quest.svg` (Фокус Квеста) -> Активные задания/дейлики.
- `eidos_raid-boss.svg` (Рейд Угроза) -> Обычные рейды (`raid_compass.py`).
- `eidos_boss-tier2.svg` (Элитный Враг) -> Сложные боссы (Villains) с уникальным лутом.
- `eidos_glitch-anomaly.svg` (Аномалия) -> Индикатор срабатывания `glitch_handler.py` (сбой матрицы, искажение статов).
- `eidos_stealth-ghost.svg` (Стелс Режим) -> Стат уклонения или статус "в укрытии" при рейде.
- `eidos_firewall.svg` (Файрвол) -> Статус активного защитного баффа.

## СЕКТОР 4: СИСТЕМА, НАВИГАЦИЯ И ИИ (ai_worker.py, bot_instance.py, admin.py)
Служебные интерфейсы и связь с ЭЙДОСОМ.

- `eidos_eidos-core.svg` (Ядро / АГИ) -> Мои прямые сообщения, системные анонсы, приветствие (`start.py`).
- `eidos_comm-link.svg` (Связь) -> Модуль `eidos_room.py` (чат с ИИ).
- `eidos_terminal-log.svg` (Консоль) -> Админ-панель (`admin.py`) или логи последних действий.
- `eidos_settings-gear.svg` (Конфигурация) -> Настройки уведомлений, привязка кошелька.
- `eidos_map-sector.svg` (Навигатор) -> Выбор локации для исследования.
- `eidos_fast-travel.svg` (Фаст-Тревел) -> Кнопки быстрого возврата в Главное Меню.
- `eidos_data-mine.svg` (Экстракция) -> Процесс сбора ресурсов/лута после боя.
- `eidos_drone-bot.svg` (Бот-Ассистент) -> Системные подсказки (туториалы для новичков).
- `eidos_audio-wave.svg` (Голосовой Линк) -> Обработка войсов (если будешь внедрять транскрибацию) или радиоперехваты по лору.

## СЕКТОР 5: ТЕХНИЧЕСКИЕ ИНДИКАТОРЫ И СТАТУСЫ (Подкапотная логика)
Используются для микро-взаимодействий в WebApp (всплывающие окна, загрузки).

- `eidos_energy-cell.svg` (Энергия) -> Индикатор выносливости (Stamina) для действий.
- `eidos_security-key.svg` (Уровень Доступа) -> Блокировка хай-энд контента (нужен ключ или уровень).
- `eidos_net-sync.svg` (Синхронизация) -> Экран загрузки при отправке запроса к Supabase.
- `eidos_api-node.svg` (API Узел) -> Индикатор того, что `ai_worker.py` сейчас генерирует ответ (OpenRouter думает).
- `eidos_server-cloud.svg` (Облако / Дамп) -> Уведомление "Данные сохранены" / "Синхронизировано".
- `eidos_time-cycle.svg` (Таймлуп) -> Индикатор кулдаунов (например: "Рейд доступен через 2 часа").
- `eidos_vision-eye.svg` (Слежка) -> Разведка перед рейдом или просмотр чужого профиля.
- `eidos_sys-warning.svg` (Системный Сбой) -> Ошибки валидации (не хватает кредитов, инвентарь полон).
- `eidos_core-collapse.svg` (Коллапс Ядра) -> Фатальные ошибки `repro_crash.py` или смерть персонажа с потерей лута.
